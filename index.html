102 Etiquetas HTML

# Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
1 <a> Define un hipervínculo. <a href="https://www.example.com">Visita Ejemplo</a> Puede tener el atributo target="_blank" para abrir en una nueva pestaña.

2 <p> Define un párrafo de texto. <p>Este es un párrafo.</p> Puede contener otros elementos en línea.
3 <img> Incrusta una imagen en la página. <img src="imagen.jpg" alt="Descripción de la imagen"> Es importante incluir el atributo alt para accesibilidad.
4 <div> Define una sección o contenedor en el documento. <div>Contenido del div</div> Generalmente usado para agrupar otros elementos.

5 <h1> Define el encabezado más importante. <h1>Este es un Encabezado</h1> Los encabezados van de <h1> a <h6>.

6 <ul> Define una lista desordenada. <ul><li>Elemento 1</li><li>Elemento 2</li></ul> Usa viñetas para los elementos.

7 <ol> Define una lista ordenada. <ol><li>Elemento 1</li><li>Elemento 2</li></ol> Usa números para los elementos.

8 <li> Define un elemento de lista. <li>Elemento de la lista</li> Debe estar dentro de <ul> o <ol>.

9 <table> Define una tabla. <table><tr><td>Celda</td></tr></table> Se usa para datos en formato tabular.

10 <tr> Define una fila en una tabla. <tr><td>Celda</td></tr> Debe estar dentro de <table>.

11 <td> Define una celda de datos en una tabla. <tr><td>Contenido de la celda</td></tr> Debe estar contenido dentro de una fila (<tr>).

12 <th> Define una celda de cabecera en una tabla. <tr><th>Cabecera</th></tr> Por defecto, el contenido se muestra en negrita y centrado.

13 <thead> Define el bloque de cabecera de una tabla. <thead><tr><th>Cabecera</th></tr></thead> Debe ir antes de <tbody> y <tfoot>.

14 <tbody> Define el bloque del cuerpo de una tabla. <tbody><tr><td>Celda</td></tr></tbody> Contiene las filas de datos principales.

15 <tfoot> Define el bloque de pie de una tabla. <tfoot><tr><td>Pie de tabla</td></tr></tfoot> Se usa para agregar una fila final a la tabla.

16 <strong> Define texto en negrita y de importancia. <strong>Este texto es importante</strong> Generalmente, el contenido se muestra en negrita.

17 <em> Define texto con énfasis. <em>Este texto tiene énfasis</em> El contenido se muestra en cursiva.

18 <br> Inserta un salto de línea. Texto antes del salto <br> Texto después del salto No tiene etiqueta de cierre.

19 <hr> Inserta una línea horizontal. <hr> Se usa para separar contenido.

20 <b> Define texto en negrita. <b>Este texto es negrita</b> Solo da formato visual, sin indicar importancia.

21 <i> Define texto en cursiva. <i>Este texto es cursiva</i> Solo da formato visual, sin indicar énfasis.

22 <blockquote> Define una cita en bloque. <blockquote>Esta es una cita.</blockquote> Por defecto, se aplica sangría al contenido.

23 <code> Define texto como código fuente. <code>console.log('Hola');</code> Se muestra en un formato monoespaciado.

24 <pre> Define texto preformateado. <pre>Texto con espacios</pre> Conserva los espacios y saltos de línea.

25 <small> Define texto de menor tamaño. <small>Este es un texto más pequeño</small> Se usa para notas al pie o contenido de menor relevancia.

26 <s> Define texto tachado. <s>Este texto está tachado</s> Indica contenido incorrecto o no relevante.

27 <sub> Define texto como subíndice. <sub>Texto subíndice</sub> Se usa para notaciones científicas o matemáticas.

28 <sup> Define texto como superíndice. <sup>Texto superíndice</sup> Se usa para exponentes o notaciones.

29 <span> Define un contenedor en línea. <span>Texto en línea</span> Se usa para aplicar estilos o agrupar texto en línea.

30 <input> Define un campo de entrada de datos. <input type="text" name="nombre"> Debe tener un tipo específico como "text", "password", etc.

31 <button> Define un botón interactivo. <button>Haz clic aquí</button> Se puede usar con eventos JavaScript.

32 <label> Define una etiqueta para un elemento <input>. <label for="nombre">Nombre:</label> El atributo "for" debe coincidir con el id del <input> asociado.

33 <form> Define un formulario HTML para la entrada de datos. <form action="/enviar" method="post">...</form> Usa atributos como "action" y "method".

34 <textarea> Define un área de texto multilínea. <textarea rows="4" cols="50">Escribe aquí...</textarea> Permite la entrada de texto en varias líneas.

35 <select> Define un menú desplegable. <select><option>Opción 1</option></select> Debe contener etiquetas <option>.

36 <option> Define una opción dentro de un <select>. <option>Opción 1</option> Se usa dentro de <select> para crear opciones seleccionables.

37 <fieldset> Agrupa elementos relacionados en un formulario. <fieldset><legend>Información personal</legend>...</fieldset> Puede incluir una leyenda usando <legend>.

38 <legend> Define una leyenda o título para un <fieldset>. <legend>Información personal</legend> Debe estar contenido dentro de <fieldset>.

39 <datalist> Define una lista de opciones predefinidas para un campo <input>. <input list="opciones"><datalist id="opciones"><option value="Opción 1"></datalist> Se usa con <input> para sugerencias de autocompletado.

40 <output> Define el resultado de un cálculo o acción. <output>Resultado aquí</output> Generalmente utilizado junto con scripts o formularios.

41 <progress> Representa el progreso de una tarea. <progress value="70" max="100">70%</progress> Los atributos "value" y "max" indican el progreso actual y el máximo.

42 <meter> Representa una medida escalar dentro de un rango conocido. <meter value="2" min="0" max="10">2 de 10</meter> Se usa para valores que se puedan medir o comparar.

43 <nav> Define una sección de navegación. <nav><a href="/">Inicio</a></nav> Se usa para enlaces de navegación.

44 <aside> Define contenido relacionado o suplementario. <aside>Esto es una barra lateral</aside> Generalmente usado para contenido lateral.

45 <article> Define contenido independiente y auto-contenido. <article><h2>Título del artículo</h2></article> Usado para artículos, blogs o entradas.

46 <header> Define la cabecera de un documento o sección. <header><h1>Encabezado</h1></header> Suele contener títulos, logos o enlaces de navegación.

47 <footer> Define el pie de página de un documento o sección. <footer><p>Derechos reservados</p></footer> Generalmente incluye información de autor o enlaces adicionales.

48 <section> Define una sección de un documento. <section><h2>Sección principal</h2></section> Usado para agrupar contenido relacionado.

49 <main> Define el contenido principal de un documento. <main><p>Este es el contenido principal</p></main> Debe ser único en la página.

50 <figure> Define contenido ilustrativo. <figure><img src="imagen.jpg" alt="Descripción"></figure> Se usa para imágenes, gráficos u otros elementos ilustrativos.

51 <figcaption> Define un pie de foto o leyenda para un <figure>. <figure><img src="imagen.jpg"><figcaption>Descripción de la imagen</figcaption></figure> Se usa dentro de un <figure>.



52 <audio> Define un contenido de audio. <audio controls><source src="audio.mp3" type="audio/mpeg"></audio> El atributo "controls" muestra controles de reproducción.

53 <video> Define un contenido de video. <video width="320" height="240" controls><source src="video.mp4" type="video/mp4"></video> Admite atributos como "controls", "width" y "height".
54 <source> Define una fuente multimedia alternativa para <audio> o <video>. <source src="audio.ogg" type="audio/ogg"> Se usa dentro de <audio> o <video>.
55 <track> Define subtítulos o descripciones para medios <audio> o <video>. <track kind="subtitles" src="subtitulos.vtt" srclang="es" label="Español"> Se usa dentro de <video> o <audio>.
56 <canvas> Define un área de dibujo gráfico mediante JavaScript. <canvas id="miCanvas" width="200" height="100"></canvas> Se usa con scripts para dibujar gráficos.


57 <svg> Define gráficos vectoriales en 2D. <svg width="100" height="100"><circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" /></svg> Se usa para gráficos escalables.
58 <embed> Inserta un contenido externo, como un video o documento. <embed src="archivo.pdf" type="application/pdf" width="600" height="400"> Usado para medios como PDFs o videos.


59 <object> Define un contenedor para recursos externos, como imágenes o documentos. <object data="imagen.svg" type="image/svg+xml"></object> Puedes incluir contenido alternativo dentro de la etiqueta.


60 <param> Define parámetros para un objeto multimedia insertado con <object>. <object data="movie.swf"><param name="autoplay" value="true"></object> Se usa con <object> para pasar valores adicionales.


61 <iframe> Define un marco embebido que carga otra página dentro de la actual. <iframe src="pagina.html" width="300" height="200"></iframe> Permite mostrar contenido externo en la página actual.


62 <link> Define la relación entre el documento actual y un recurso externo. <link rel="stylesheet" href="estilos.css"> Generalmente usado para cargar hojas de estilo CSS.


63 <meta> Proporciona metadatos sobre el documento HTML. <meta charset="UTF-8"> Utilizado para especificar el conjunto de caracteres o información de SEO.


64 <style> Define estilos CSS internos para el documento. <style>body { font-size: 16px; }</style> Se usa para definir reglas CSS dentro de un documento HTML.


65 <title> Define el título del documento, que aparece en la barra del navegador. <title>Mi página web</title> Debe estar dentro de la etiqueta <head>.


66 <head> Contiene metadatos y enlaces al contenido externo del documento. <head><title>Título</title></head> Incluye etiquetas como <title>, <meta>, <link>, etc.
67 <body> Define el cuerpo del documento, que contiene todo el contenido visible. <body><h1>Hola Mundo</h1></body> Debe incluir todo el contenido visible en la página web.
68 <html> Define el contenedor raíz de un documento HTML. <html lang="es"><head>...</head><body>...</body></html> Contiene las etiquetas <head> y <body>.


69 <base> Especifica la URL base para todas las URLs relativas en un documento. <base href="https://ejemplo.com/"> Se coloca dentro de la etiqueta <head>.


70 <basefont> (obsoleto) Define el tamaño, color y fuente base del texto del documento. <basefont size="4" color="red"> Obsoleta, reemplazada por CSS para estilos de texto.


71 <noscript> Define un contenido alternativo para navegadores que no soportan JavaScript. <noscript>Su navegador no soporta JavaScript</noscript> Se muestra cuando el script no es compatible o está desactivado.


72 <script> Define un bloque de código JavaScript embebido o vinculado externamente. <script src="script.js"></script> Usado para ejecutar scripts dentro de la página web.


73 <del> Representa texto que ha sido eliminado de un documento. <del>Este texto ha sido eliminado</del> El texto aparece con una línea a través de él.


74 <ins> Representa texto que ha sido insertado en un documento. <ins>Este es un nuevo texto</ins> El texto insertado aparece subrayado.


75 <small> Hace que el texto aparezca más pequeño que el texto circundante. <small>Este es un texto más pequeño</small> Se usa para disclaimers o aclaraciones.


76 <mark> Resalta texto como si estuviera marcado con un resaltador. <mark>Este texto está resaltado</mark> Usado para enfatizar o resaltar texto importante.


77 <sub> Define texto como subíndice, apareciendo más abajo que el texto normal. <p>H<sub>2</sub>O es agua</p> Usado en fórmulas matemáticas o químicas.


78 <sup> Define texto como superíndice, apareciendo más arriba que el texto normal. <p>E=mc<sup>2</sup></p> Usado en fórmulas matemáticas o químicas.


79 <bdi> Permite aislar un fragmento de texto que podría tener una dirección de texto diferente. <bdi>Usuario123</bdi> ha iniciado sesión Usado en idiomas con direcciones de texto mixtas.


80 <bdo> Controla explícitamente la dirección del texto en su contenido. <bdo dir="rtl">Este texto se mostrará de derecha a izquierda</bdo> Útil en idiomas que leen de derecha a izquierda.


81 <wbr> Define una posición en la que el navegador puede interrumpir una línea de texto si es necesario. <p>Este es un texto muy largo<wbr> que puede romperse.</p> Útil para controlar el flujo de texto en líneas largas.


82 <data> Vincula un contenido con un valor legible por máquina. <data value="12345">Número de serie</data> Se usa para representar datos legibles por humanos y máquinas.


83 <time> Representa un valor de fecha y/o hora. <time datetime="2023-12-01T12:00">1 de diciembre de 2023, 12:00 PM</time> Es compatible con los formatos de fecha y hora de HTML5.


84 <progress> Representa el progreso de una tarea en ejecución. <progress value="70" max="100">70%</progress> Visualiza el porcentaje de finalización de una tarea.


85 <meter> Representa una medición dentro de un rango conocido. <meter value="0.7">70%</meter> Se usa para representar valores en un rango específico.


86 <output> Representa el resultado de un cálculo. <output name="resultado">42</output> Generalmente usado en combinación con formularios y scripts.


87 <picture> Contenedor para diferentes versiones de una imagen según el tamaño de la pantalla. <picture><source srcset="imagen-grande.jpg" media="(min-width: 800px)"><img src="imagen-pequeña.jpg" alt="Descripción"></picture> Útil para crear imágenes responsivas.


88 <summary> Define un encabezado visible para un elemento <details>. <details><summary>Más información</summary><p>Texto oculto</p></details> Permite que los detalles adicionales se expandan o contraigan.


89 <details> Define un bloque que el usuario puede mostrar u ocultar. <details><summary>Ver más</summary><p>Texto adicional aquí.</p></details> Se usa para proporcionar información adicional que el usuario puede ver bajo demanda.


90 <dialog> Representa un cuadro de diálogo o ventana emergente. <dialog open>Este es un cuadro de diálogo</dialog> Permite crear ventanas modales o no modales.


91 <li> Define un elemento de lista. <li>Elemento de la lista</li> Debe estar dentro de <ul> o <ol>.


92 <strong> Define texto en negrita y de importancia. <strong>Este texto es importante</strong> Generalmente, el contenido se muestra en negrita.


93 <i> Define texto en cursiva. <i>Este texto es cursiva</i> Solo da formato visual, sin indicar énfasis.


94 <code> Define texto como código fuente. <code>console.log('Hola');</code> Se muestra en un formato monoespaciado.


95 <pre> Define texto preformateado. <pre>Texto con espacios</pre> Conserva los espacios y saltos de línea.


96 <small> Define texto de menor tamaño. <small>Este es un texto más pequeño</small> Se usa para notas al pie o contenido de menor relevancia.


97 <object> Define un contenedor para recursos externos, como imágenes o documentos. <object data="imagen.svg" type="image/svg+xml"></object> Puedes incluir contenido alternativo dentro de la etiqueta.


98 <param> Define parámetros para un objeto multimedia insertado con <object>. <object data="movie.swf"><param name="autoplay" value="true"></object> Se usa con <object> para pasar valores adicionales.
